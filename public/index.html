General Routing

Path HTTP Methods Description
/product GET Retrieves a list of products.
/update POST Creates a new product.
/products/:id GET Retrieves details of a specific product.
/products/:id PATCH Updates details of a specific product.
/products/:id DELETE Deletes a specific product.
